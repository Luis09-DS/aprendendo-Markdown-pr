# projeto de Exemplos
## Linguagem Utilizada
### Variáveis

```
Utilize const ou let para declaração de variavel
```


```
//ruim
var nome = 'Diogo';

//bom
let nome = 'Diogo';
```
---
### Definição de Funções
```
Evite o uso de Else
```

```
//ruim
if (a == b) {
 console.log('Iguais');
} else {
 console.log ('Diferentes');
}
```

```
//bom
if (a == b) 
 console.log('Iguais');

if (a != b)
 console.log ('Diferentes');
```
